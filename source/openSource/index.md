- [jj-plan](https://github.com/yigegongjiang/jj-plan)（CLI + Cloudflare Worker）：个人 Spec / Task 追踪，数据落 Cloudflare D1，CLI 与浏览器双端访问。
- [cli-prompt-launcher](https://github.com/yigegongjiang/cli-prompt-launcher)（CLI / Bun）：`jj` 命令以预设 scene prompt 启动 Claude Code 或 Codex，支持交互 / 文本 / 流式模式。
- [jj-skills](https://github.com/yigegongjiang/jj-skills)（npx skills 源）：个人多技能仓库，自定义 skills 供 `npx skills` 一键安装到 Claude Code / Codex 等。
- [Speedo](https://github.com/yigegongjiang/Speedo)（iOS app）：通过画中画浮窗实时显示设备上下行网速。
- [export-chromium-history](https://github.com/yigegongjiang/export-chromium-history)（CLI / Python）：读取 Chromium 系浏览器 History SQLite 库，导出为 Safari 可导入的 JSON，做跨浏览器迁移。
- [JapanMemory](https://github.com/yigegongjiang/JapanMemory)（Mac app）：日语五十音 / 单词速记，定时循环随机展示加强记忆。
- [HLVPaste](https://github.com/yigegongjiang/HLVPaste)（Mac app）：轻量级文本输入框，配合 LaunchBar 唤起，自动写入粘贴板并支持 iPhone 同步读取。
- 中文词语纠错（[说明](https://www.yigegongjiang.com/2024/SwiftZhTextCorrect/)）：[HLVSentence](https://github.com/yigegongjiang/HLVSentence)（CLI） + [HLVZhCorrect](https://github.com/yigegongjiang/HLVZhCorrect)（Mac app）。
- [HLVFileDump](https://github.com/yigegongjiang/HLVFileDump)（CLI）：文件类型识别，通过多位置截取解码以低开销判断文本文件。
